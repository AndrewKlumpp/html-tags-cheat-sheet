HTML Tags Cheat Sheet

Headings

Headings from `h1` to `h6` identify blocks and sections. Every page must have one `h1`.

Paragraphs

Paragraphs of text go in 'p' tags. Each 'p' tag contains the text of a single paragraph between the opening and closing tag. This is a better way to separate paragraphs than adding a bunch of 'br' tags after each paragraph. Very long paragraphs will fill the width of a page if CSS styles are not used to control the width. If each sentence of the paragraph is on a new line in VS Code it will making reading the code much easier.

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in `ul` tags. Each list item goes in an `li` tag.

Types of Lists:

  • Unordered Lists ('ul')
  • Ordered Lists ('ol')

Ordered Lists

Ordered lists are numbered and go in `ol` tags. Each list item goes in an `li` tag.
